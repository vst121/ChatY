# ChatY User Guide

Welcome to ChatY, your modern chat application! This guide will help you get started with using ChatY for real-time messaging and collaboration.

## Getting Started

### Accessing ChatY

1. Open your web browser and navigate to the ChatY application URL
2. If you don't have an account, you'll need to register
3. If you have an account, log in with your credentials

### System Requirements

- Modern web browser (Chrome, Firefox, Safari, Edge)
- Internet connection for real-time features
- JavaScript enabled

## User Profile

### Setting Up Your Profile

1. After logging in, click on your profile picture or name in the top navigation
2. Update your display name, profile picture, and status
3. Set your privacy preferences

### Status Settings

- **Online**: You're active and available
- **Away**: You're temporarily away
- **Offline**: You're not using the app
- **Custom**: Set a custom status message

## Chats and Messaging

### Starting a New Chat

1. Click the "New Chat" button in the sidebar
2. Search for a user by name or username
3. Select the user to start a conversation

### Creating Group Chats

1. Click "New Group" in the sidebar
2. Add participants by searching for users
3. Set a group name and optional description
4. Choose group settings (public/private)

### Sending Messages

1. Type your message in the input field at the bottom of the chat
2. Press Enter or click the send button
3. For multi-line messages, use Shift+Enter

### Message Features

#### Text Formatting

- Use markdown for basic formatting (bold, italic, etc.)
- Mention users with @username
- Use emojis and stickers

#### File Sharing

1. Click the attachment button (ðŸ“Ž) next to the message input
2. Select files from your device
3. Add optional captions
4. Send the files

#### Media Messages

- **Images**: Drag and drop or click to upload
- **Videos**: Upload video files (supported formats: MP4, WebM)
- **Voice Notes**: Click the microphone button to record
- **GIFs**: Search and send animated GIFs

#### Message Reactions

1. Hover over a message
2. Click the emoji button that appears
3. Choose from common reactions or add custom ones

#### Replying to Messages

1. Click the reply button on any message
2. Type your reply
3. The original message will be shown as context

#### Editing Messages

1. Click the three dots menu on your message
2. Select "Edit"
3. Make changes and save

#### Deleting Messages

1. Click the three dots menu on your message
2. Select "Delete"
3. Confirm deletion

#### Pinning Messages

1. Click the pin icon on important messages
2. Pinned messages appear at the top of the chat

## Chat Organization

### Folders

Organize your chats into folders:

1. Right-click on a chat in the sidebar
2. Select "Move to Folder"
3. Choose an existing folder or create a new one
4. Common folders: Work, Personal, Favorites

### Search

Find messages and chats quickly:

1. Use the search bar at the top
2. Search by keywords, usernames, or dates
3. Use filters: Unread, Mentions, Files, Links

## Group Management

### Admin Tools

If you're a group admin:

- **Add Members**: Invite new users to the group
- **Remove Members**: Remove users from the group
- **Change Roles**: Promote/demote members
- **Group Settings**: Modify group name, description, and privacy

### Group Types

- **Public Groups**: Anyone can join
- **Private Groups**: Invite-only
- **Announcement Channels**: Only admins can post

## Privacy and Security

### Privacy Controls

1. Go to Settings > Privacy
2. Control who can:
   - Message you
   - Add you to groups
   - See your profile information
   - See your online status

### Blocking Users

1. Open a chat with the user
2. Click their profile picture
3. Select "Block User"
4. Confirm the action

### Disappearing Messages

1. In chat settings, enable disappearing messages
2. Choose time interval (24 hours, 7 days, etc.)
3. Messages will automatically delete after the set time

## Tasks and Polls

### Shared Tasks

1. In a chat, click the task button
2. Create a new task
3. Assign to participants
4. Set due dates and priorities

### Creating Polls

1. Click the poll button in chat
2. Add question and options
3. Set poll duration
4. Participants can vote anonymously or publicly

## Multi-Device Usage

### Adding Devices

1. Log in to ChatY on multiple devices
2. Messages sync automatically across all devices
3. Your chat history is available everywhere

### Device Management

1. Go to Settings > Devices
2. View all logged-in devices
3. Remove devices you no longer use

## Themes and Customization

### Changing Themes

1. Click your profile picture > Settings
2. Select Appearance
3. Choose between Light, Dark, or Auto mode

### Custom Backgrounds

1. In chat settings, select "Change Background"
2. Upload a custom image or choose from presets

## Troubleshooting

### Connection Issues

- Check your internet connection
- Try refreshing the page
- Clear browser cache and cookies

### Message Not Sending

- Check file size limits for attachments
- Ensure you're not blocked by the recipient
- Try sending as a text message instead

### Notifications Not Working

- Check browser notification permissions
- Ensure ChatY is not muted in your browser
- Verify notification settings in the app

## Keyboard Shortcuts

- `Ctrl + /`: Show keyboard shortcuts
- `Ctrl + K`: Quick search
- `Ctrl + N`: New chat
- `Enter`: Send message
- `Shift + Enter`: New line
- `Ctrl + B`: Bold text
- `Ctrl + I`: Italic text
